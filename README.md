# Data Structure & Algorithm Visualizer

An interactive desktop app that visualizes core data structures in real time — built with Python and Tkinter.

## Supported Structures

| Structure | Operations |
|---|---|
| Binary Search Tree (BST) | Insert, delete, search, animated pre-order traversal |
| Linked List | Insert at end, delete by value, search by position |
| Stack | Push, pop |
| Queue | Enqueue, dequeue |
| Min Heap | Insert, extract-min |

## Features

- Live canvas rendering — watch nodes appear, move, and disappear as you operate
- Highlight animation on search — found node is visually marked
- Adjustable animation speed via slider
- Status bar feedback on every operation
- Dropdown to switch structures without losing others' state

## Run It

```bash
pip install numpy  # only external dependency
python "data strcuture proj.py"
```

No installation needed beyond Python 3 and Tkinter (included in standard library).
